# social-network
A lightweight social networking platform built with PHP, HTML, and CSS, designed to support user profiles, social interactions, and content sharing in a traditional server-rendered web architecture.

Technology Stack: PHP: Core server-side language handling routing, business logic, and database interaction, MySQL: Relational database for users, posts, relationships, and metadata, HTML5: Semantic markup for page structure, CSS3: Custom styling, layout, and responsive design, Apache: Web Server, phpMyAdmin: Database administration, Git: Version control
